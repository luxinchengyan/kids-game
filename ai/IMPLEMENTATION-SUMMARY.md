# 童梦飞船 · 智趣成长

## 概述

本文档总结了基于通用游戏框架为各个学习岛实现的游戏。所有打地鼠游戏都使用统一的 `WhackAMole` 框架，通过不同的主题配置实现多样化学习内容。

**路由**: `/games/pinyin`

**整体认读音节** (5个):
- zhi (蜘蛛🕷️), chi (吃饭🍜), shi (狮子🦁)

## 🎯 游戏特性

### 通用特性（所有游戏共享）

✅ **三难度级别**: 入门、进阶、挑战  
✅ **连击系统**: 连续正确获得倍数分数  
✅ **即时反馈**: 正确/错误动画提示  
✅ **语音提示**: TTS朗读题目  
✅ **统计追踪**: 分数、正确率、最大连击  
✅ **星级评价**: 根据正确率给予1-3星  
✅ **进度保存**: 使用 `useGameCompletion` hook  
✅ **数据分析**: 集成 `track` 统计系统  

### 主题特性

| 主题 | 特殊功能 |
|-----|---------|
| **拼音** | 声母/韵母/音节分类，混淆音提示 |
| **数学** | 数字+形状+加法多种题型 |
| **英语** | 英语发音 (en-US)，双语对照 |
| **汉字** | 大字显示，文化背景示例 |

## 🚀 使用示例

### 访问游戏

1. **拼音冒险岛**:
   ```
   首页 → 拼音冒险岛 → 拼音打地鼠（新版）
   或直接访问: /games/pinyin/whack-a-mole-generic
   ```

2. **数字小镇**:
   ```
   首页 → 数字小镇 → 数学打地鼠
   或直接访问: /games/math/whack-a-mole
   ```

3. **英语游乐园**:
   ```
   首页 → 英语游乐园 → 英语打地鼠
   或直接访问: /games/english/whack-a-mole
   ```

4. **故事王国**:
   ```
   首页 → 故事王国 → 汉字打地鼠
   或直接访问: /games/stories/whack-a-mole
   ```

### 游戏流程

```
开始界面
  ↓ 选择难度 (入门/进阶/挑战)
游戏界面
  ↓ 显示目标 (语音+文字)
  ↓ 地鼠出现 (随机位置)
  ↓ 点击地鼠 (判断对错)
  ↓ 反馈动画 (正确✅/错误❌)
  ↓ 进入下一回合
结果界面
  ↓ 显示星级 (1-3星)
  ↓ 显示统计 (分数/正确率/连击)
  ↓ 再玩一次 / 返回首页
```

## 💡 技术亮点

### 1. 架构设计

```
通用框架 (一次开发)
    ↓ 注入 ↓
主题配置 (多次复用)
    ↓ 组合 ↓
具体游戏 (快速生成)
```

### 2. 类型安全

- 完整的 TypeScript 类型定义
- 接口约束确保主题配置正确
- 编译时错误检查

### 3. 性能优化

- React.lazy 懒加载
- 定时器自动清理
- 避免不必要的重渲染

### 4. 可维护性

- 单一职责原则
- 配置与逻辑分离
- 易于测试和调试

## 📝 开发指南

### 添加新游戏到现有学习岛

```typescript
// 1. 在对应的 index.ts 中注册
import { registerGame } from '../registry';
import { lazy } from 'react';

const NewGame = lazy(() => import('./NewGame'));

registerGame({
  id: 'new-game',
  name: '新游戏',
  icon: '🎮',
  description: '游戏描述',
  path: '/games/pinyin/new-game',
  component: NewGame,
  category: 'pinyin',
  themeId: 'pinyin',
  minAge: 4,
  maxAge: 8,
});
```

### 创建新学习岛

```typescript
// 1. 创建主题目录
src/games/new-subject/

// 2. 创建主题中心组件
src/games/new-subject/NewSubjectThemeHub.tsx

// 3. 创建游戏注册文件
src/games/new-subject/index.ts

// 4. 在 App.tsx 中导入
import './games/new-subject';
```
